# 01 - Hello World Markdown Artifact

This Agent App creates `output/hello-world.md` containing:

```markdown
Hello, **World**!
```

It then designates that file as the `hello-world` Artifact. Karta announces the
Artifact in the conversation and lists it under **Artifacts**. Open it from
either location; close the right-hand canvas and reopen it from the list.

Deploy the folder as an ordinary Karta Agent App, start a conversation, and ask
it to create the Hello World Artifact.
