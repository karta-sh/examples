# 02 - Generated Report Artifact

This Agent App turns two small source files into one Markdown report and
designates exactly one Artifact. The report is the rendered entrypoint; the
notes and metrics travel with that immutable Artifact version as evidence.

Deploy the folder as an ordinary Karta Agent App, start a conversation, and ask
it to create the sample launch report. Karta announces the result in the
conversation and keeps it in the **Artifacts** list for reopening.

This example adds a multi-file Artifact and makes the distinction between the
rendered output and its supporting evidence concrete.
