# 03 - Data Table Artifact

This Agent App produces one CSV Artifact. Karta's built-in file renderer opens
the data as a bounded table, so the output needs no custom presentation code.

Deploy the folder, start a conversation, and ask for the sample inventory
table. Open it from either the **Artifact ready** card or the persistent
**Artifacts** list, close the canvas, and reopen it.

This example shows an Artifact that is data plus Karta-provided rendering.
